# Dracula for [Miniflux](https://miniflux.app/)

> A dark theme for [Miniflux](https://miniflux.app/)

![Screenshot](./miniflux-dracula.png)

## Install

* unpack css .zip file
* copy css file content
* paste in Miniflux > Settings - Custom CSS textarea
